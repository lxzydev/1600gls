A car mod for **My Winter Car** that modifies the stock Sorbet 1600 with a custom GLS trim.

Original mod by Dom FULL credit to Dom. This version was decompiled and fixed by lxzy after the original was removed from Nexus.
This mod was originally hosted on Nexus and has since been taken down. This is a preservation release with full credit to the original author. If Dom wants this taken down, reach out and it will be removed.

---

## Features

### Exterior
- Multiple exterior mesh replacements
- Redesigned hood
- Custom front and rear light housings
- New front door panels
- Redesigned side mirrors
- Custom boot lid / hatch
- Custom door window glass
- Custom transparent decal/sticker system on body and rear glass
- Stock chassis mesh, mudflaps, registration plate, and hatch sticker removed and replaced

### Lighting
- Custom indicator lights
- Redesigned headlight beams 
- Custom reverse light assembly
- Beam marker lights (left & right)

### Interior
- Optional custom steering wheel
- Woodpearls dashboard accessory

### Mod Settings
- Steering Wheel: Stock / New custom wheel
- Woodpearls: ON / OFF
- Changes apply on save & reload

---

## Installation

1. Install MSCLoader if you havent already
2. Drop `GlsSorbet1600.dll` into your `Mods` folder
3. Place `sob.unity3d` and `sorbet_sticks.png` into `Mods/Assets/GlsSorbet1600/`
4. Launch the game
